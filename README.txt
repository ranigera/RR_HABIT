This repository contains the pilot data and the Matlab and R code for the experimental task and the statistical analysis of the pilot data of the project: 

"Does anxiety moderate training duration effects on habits in humans?  Determining the effects of anxiety on the experimental induction of habits in an instrumental outcome devaluation task"

Folder structure:

Pilot data -contains the compiled databases, the analysis R code and the preregistration documents

Experimental task - contains the Matlab code and the specific instruction of the experimental task
